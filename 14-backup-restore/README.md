# 14 - Backup & Restore

## Concepts covered
- SQLite: file-based backup (it's literally just a file)
- PostgreSQL: pg_dump / pg_restore
- Docker Compose setup to run a real PostgreSQL locally for practicing this

## Files
- `sqlite-backup.js` — programmatic SQLite backup using the .backup() API
- `postgres-backup-commands.md` — the real pg_dump/pg_restore commands
- `docker-compose.yml` — spin up local Postgres to practice against
