# 01 - Fundamentals

## Concepts covered
- Creating a table, inserting data
- SELECT, WHERE, ORDER BY, LIMIT
- Basic UPDATE and DELETE

## Files
- `schema.sql` — creates and seeds a `books` table
- `queries.sql` — every query below, with comments explaining each
- `run.js` — executes schema.sql then queries.sql and prints results

## Run it
```bash
npm install
node run.js
```
