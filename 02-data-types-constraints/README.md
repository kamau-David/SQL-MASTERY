# 02 - Data Types & Constraints

## Concepts covered
- Common column types (INTEGER, TEXT, REAL, BOOLEAN, DATE)
- Constraints: NOT NULL, UNIQUE, CHECK, DEFAULT
- PRIMARY KEY vs FOREIGN KEY
- Why constraints matter: they stop bad data at the database level, not just in your app code

## Files
- `schema.sql`
- `test-constraints.sql` — deliberately tries invalid inserts to prove constraints work
- `run.js`

## Run it
```bash
npm install
node run.js
```
