# 09 - Views & Stored Procedures

## Concepts covered
- Views: saved, reusable queries that behave like virtual tables
- Stored procedures/functions - PostgreSQL-only (SQLite has no procedural language)
- When to use each

## Files
- `views-demo.js` — views (works in SQLite)
- `postgres-functions.sql` — stored procedure/function examples (requires real PostgreSQL)

## Run it
```bash
npm install
node views-demo.js
# postgres-functions.sql requires a running PostgreSQL instance (see folder 14 for Docker setup)
```
